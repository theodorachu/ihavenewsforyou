I used https://blog.miguelgrinberg.com/post/the-flask-mega-tutorial-part-i-hello-world to learn Flask.


To Create the Database:
./db_create.py


To Run:
./runFlaskServer.py


To Use the Database:
We are using SQL-Alchemy. See docs: http://www.sqlalchemy.org/

Virtual Environment:
I'm using a virtual environment so you don't have to download anything. Just remember to activate it via 'source Virtualenv/bin/activate' and deactivate it by typing 'deactivate'

Weird files, but you have to know about them:
1) runtime.txt - specifies python version for heroku
2) Procfile - also for heroku
